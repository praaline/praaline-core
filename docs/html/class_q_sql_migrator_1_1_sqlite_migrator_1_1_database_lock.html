Detailed Description

The DatabaseLock try to lock a database for this QSqlMigrator process. You have to check if you got the lock ( see example with operator bool ).

The lock will be done with a lock-file in the same folder where the db is locatded.

The lock will automatically released on deconstruction.

Example:

unsigned int timeOutToGetLock = 60; context c( ... ); DatabaseLock lock(c, timeOutToGetLock); if(lock) { // do mirgration stuff }
