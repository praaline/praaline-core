Detailed Description

The QSqlMigratorService class is used to control all relevant features of the QSqlMigrator. The User should use this service within it's own applications.

context c; if( ! missingMigrations(c).empty() ) exit(1); if( ! unappliedMigrations(c).empty() ) { DatabaseLock lock(c); if(lock) applyAll(c);

// or exit(1); } // here all migrations are applied
