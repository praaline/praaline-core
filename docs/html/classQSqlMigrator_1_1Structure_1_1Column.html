Detailed Description

value object representing the column-structure.

Columns are default NULL (as in SQLite, MySQL and PostgreSQL). According to the SQL standard, PRIMARY KEY should always imply NOT NULL. UNIQUE columns can contain several NULL values (as in SQLite, MySQL and PostgreSQL) AutoIncrement is special: every DBMS handles some sort of auto increment in its own way.
